catvehicle - A ROS package for autonomous vehicle applications
==================================================================

.. image::  https://raw.githubusercontent.com/jmscslgroup/catvehicle/melodic_gazebo-9/docs/source/favicon.png
   :width: 500px
   :align: left

**catvehicle**, is a ROS package for autonomous vehicle applications.

|

Contributing to the project
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Feel free to submit an `issue <https://github.com/jmscslgroup/catvehicle/issues/new/choose>`_
or send us an `email <mailto:rahulbhadani@email.arizona.edu>`_. 
If you like to contribute to this project, please fork this repository to your GitHub account,
create a new branch for yourself and send a pull request for the merge. After reviewing the changes,
we will decide if this is a good place to add your changes.
Your help to improve **catvehicle** is highly appreciated.
